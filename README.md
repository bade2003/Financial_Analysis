# Financial_Analysis
Developed an interactive Financial Analysis Dashboard using Power BI and Python to analyze and visualize key financial metrics such as Sales, Profit, Income, Price, Turnover, and Market Trends. The project involved collecting and preparing financial datasets, performing data cleaning, transformation, and exploratory data analysis (EDA) to ensure accuracy and usability of the data.

Implemented various data visualization techniques including line charts, bar charts, scatter plots, and pie charts to identify patterns, trends, and correlations between financial variables such as Sales vs Profit and Price vs Volume. Created time-series analysis to monitor financial performance across different periods using date hierarchies (Year, Month, Day).

Integrated Python scripts inside Power BI using libraries such as Pandas and Matplotlib to generate advanced visualizations and perform statistical analysis. Designed the dashboard with interactive filters, slicers, and drill-down capabilities, allowing users to explore financial data dynamically by date, region, and financial metrics.

The dashboard enables stakeholders to track KPIs, evaluate financial performance, identify revenue patterns, and support strategic business decision-making. This project demonstrates strong skills in data analysis, data visualization, business intelligence, and dashboard development.
